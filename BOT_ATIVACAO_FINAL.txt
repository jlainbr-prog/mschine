🔍 TROUBLESHOOTING
================================================================================

❌ ERRO: "RPC not configured"
✅ SOLUÇÃO: Verifique if RPC_ETH e RPC_BSC estão corretos no .env
   - TestE: curl https://[sua-rpc]/
   - Considere mudar de provider se essa falhar

❌ ERRO: "Bot não encontra oportunidades"
✅ SOLUÇÃO: 
   - Diminua MIN_PROFIT_USD (comece com 5-10 USD)
   - Verifique se os pools têm liquidez suficiente
   - Check network delays (RPC pode estar lento)

❌ ERRO: "Package-lock.json required"
✅ SOLUÇÃO: Já resolvido! npm install foi executado

❌ ERRO: Node não acessa .env
✅ SOLUÇÃO: 
   - Instale dotenv: npm install dotenv
   - Verifique if .env está no diretório raiz
